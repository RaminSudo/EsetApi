# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/RaminSudo/EsetApi.git
cd EsetApi
chmod +x eset.sh
./eset.sh install
./eset.sh 


# For Auto Launch:
chmod 777 auto.sh
tmux
./auto.sh
# End ;)
```
